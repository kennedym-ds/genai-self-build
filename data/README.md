# GenAI Self-Build Workshop Files

This data folder contains sample datasets for the workshops.

## Suggested Contents

### corpus/
- `sample_text.txt` - General text for tokenization and embedding training
- `simple_wikipedia.txt` - Simple Wikipedia excerpts

### documents/
- `knowledge_base/` - Short documents for RAG demo (FAQ, articles, etc.)

## Data Sources

Consider using:
- Simple Wikipedia: https://simple.wikipedia.org
- News articles (public domain)
- Technical documentation
- FAQ collections

Keep documents short (1-2 paragraphs) for workshop efficiency.
